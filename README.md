## <img src="https://em-content.zobj.net/source/microsoft-teams/363/hammer_and_wrench_1f6e0.gif" width="30"> Modular Architecture

*Flow uses a unique **Core-to-Plugin** bridge design — decoupling the playback engine from content sources.*

</div>

<br>

<table align="center">
<tr>
<td width="42%" align="center" style="background: linear-gradient(135deg, #0a1628, #0d2847); border-radius: 12px; padding: 30px;">

### ⚙️ The Engine (Core)

` Handles everything the user sees and hears. `

<img src="https://img.shields.io/badge/UI_Rendering-007BFF?style=flat-square" />
<img src="https://img.shields.io/badge/Media_Session-007BFF?style=flat-square" />
<img src="https://img.shields.io/badge/Audio_Focus-007BFF?style=flat-square" />

class FlowEngine {
    UI.render()

## <img src="https://em-content.zobj.net/source/microsoft-teams/363/rocket_1f680.gif" width="30"> Installation

*Two simple phases to get Flow running.*

</div>

<br>

<table align="center">
<tr>
<td width="48%" style="background: linear-gradient(135deg, #0a1628, #0d2847); border-radius: 12px; padding: 30px;">

### Phase 1 — Install Flow App

1. Go to the **[Flow Releases](https://github.com/Garvittt-API/Flow-releases/releases)** page
2. Download the latest `.apk` file
3. Allow **"Install from unknown sources"** if prompted
4. Open & install the Flow app

<a href="https://github.com/Garvittt-API/Flow-releases/releases">
  <img src="https://img.shields.io/badge/📥_DOWNLOAD_APK-007BFF?style=for-the-badge&logo=android&logoColor=white" alt="Download APK" />
</a>

</td>
<td width="4%"></td>
<td width="48%" style="background: linear-gradient(135deg, #1a0a2e, #2d1b4e); border-radius: 12px; padding: 30px;">

### Phase 2 — Add Extensions

1. Download from the **[Extensions Vault](https://github.com/Garvittt-API/Flow-extension/releases/tag/extension)**
2. Open Flow → tap **Settings** (⚙️ gear icon)
3. Scroll to **Extensions** → tap **Plus (+)**
4. Select the `.eapk` file to activate

<a href="https://github.com/Garvittt-API/Flow-extension/releases/tag/extension">
  <img src="https://img.shields.io/badge/🔌_GET_EXTENSIONS-7B2FF7?style=for-the-badge&logo=plug&logoColor=white" alt="Get Extensions" />
</a>
